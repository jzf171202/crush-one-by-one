# 各个击破
这是一个针对Android核心知识模块进行的源码分析实例项目，目的是为了巩固知识并深入学习。

核心功能点：
- 整合MVP+Retrofit2.0+Rxjava2.0+DataBinding；
- 规避MVP内存泄露问题；
- 网络请求状态码动态配置业务逻辑；
- API 23后权限动态申请引入。
- EasyGuideView新手引导开源库引入示例
- synchronized同步锁机制实战
- 动态代理机制实战
- Bitmap采样率压缩方案实战
- View的事件传递机制
- View的绘制流程
- Hybrid混合开发框架方案(一)


—— 现有源码有删减，可从git版本历史记录里查看对应功能点。
